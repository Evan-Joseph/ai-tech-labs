# 实验二：基于 Adaboost 及 SVM 的人脸识别

本目录为实验二的代码与资料占位，后续将补充实现与数据链接。

- 实验目的：
  - 掌握 Adaboost 与 SVM 的基本原理与差异，理解强分类器的构建流程。
  - 在标准人脸数据集上实现与对比两种方法的识别性能。
- 实验内容：
  - 特征工程（如HOG/LBP等）与数据预处理。
  - 基于 Adaboost 的人脸分类器训练与评估。
  - 基于 SVM（线性/核SVM）的训练与评估。
  - 指标：准确率、混淆矩阵、ROC/PR（可选）。
- 实验条件：Python 3.11；numpy、scikit-learn、matplotlib、seaborn、pandas；可根据数据集大小选择缓存目录。
- 实验要求：可复现实验；代码结构清晰；结果图表保存至 `assets/`。
- 实验步骤：数据准备 → 特征工程 → 模型训练 → 评估与可视化 → 小结。
- 实验报告：简述方法、实验设置、结果对比与分析、结论与反思。

目录建议：
- `experiments/`：可运行脚本与一键 `run_all.py`
- `src/`：算法与工具实现
- `assets/`：图与表（不提交大数据）
